Web Programming - I

Welcome to Timer & To-Do List project! This web application is designed to enhance productivity by organizing tasks efficiently. The site is divided into three main sections: Regular Notes: This section allows users to create and manage notes in a straightforward manner. Users can jot down important tasks or reminders with ease. Timed Notes: In this section, users can create notes with a specified time limit. Once the time is up, the application will notify the user, ensuring that important tasks are completed on time. Deleted Notes: This section keeps a record of previously deleted notes. Users can review and restore any deleted notes if needed, providing a safety net for accidentally removed tasks.
